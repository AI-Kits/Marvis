# Marvis

Jarvis for my Money，股票、基金、理财的数据小助手

# 投基战略

## 投资组合

三分之一稳健配置、三分之一优选混合（大盘指数）、三分之一行业指数

## 投资札记

- 不追高
- 时刻动态平衡，保证手中存在现金流；非绝望不满仓
